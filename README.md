# Sea Cucumber Farming Website 🌊

**AQUA BLOOM INTERNATIONAL PVT LTD** — Static website built with React, Tailwind CSS, Flowbite, and AOS animations.  

This website showcases **sustainable sea cucumber farming**, its economic potential, and environmental benefits. It is a modern, responsive, and animated single-page website.

---

## 🌟 Features

- Fully **responsive design** for desktop, tablet, and mobile  
- **Modern UI** with Tailwind CSS + Flowbite components  
- **Animated sections** using AOS and Framer Motion  
- **Google Fonts**: Poppins (headings), Inter (body)  
- **Single-page layout** with smooth scrolling  
- **Reusable components** for hero, buttons, cards, and sections  

---

## 🛠 Tech Stack

| Technology          | Purpose                                         |
